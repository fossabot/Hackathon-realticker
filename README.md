🚀 RealTicker – AI-Powered Stock Insights Platform

RealTicker is a full-stack web application developed as part of a hackathon project. It helps users explore stock trends through interactive charts and provides basic AI-assisted insights to support understanding of market movement.

The application uses a React frontend and a FastAPI backend, with optional HuggingFace integration for generating analysis. Mock data is included so the project can run completely offline for demo purposes.

This project focuses on practical full-stack development, REST API implementation, data visualization, and simple AI integration.

✅ Key Features

Displays Top 10 stocks with sorting based on volume, growth percentage, or market cap

Visualizes 6 months of historical stock prices using interactive charts

Generates AI-based investment insights (falls back to rule-based analysis if LLM is unavailable)

Responsive user interface built with React and Bootstrap

REST APIs implemented using FastAPI

Loading indicators and basic error handling for better user experience

Mobile-friendly layout

Includes mock stock data for testing and demonstrations
